VJI Embark

VJI Embark is a free remote clip launcher for Resolume Arena and Avenue VJ software.

Overview

  • 4 layers
  • opacity, bypass, assignable layer invert
  • immediate blend mode changes
  • "flipvert" to quickly toggle
  • clip arming, punch-ins
  • multi-touch, faders and blend modes can be swiped across
